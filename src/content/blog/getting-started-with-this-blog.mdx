# Getting Started with This Blog

Welcome to Confero! This guide will help you navigate and make the most of this multi-space blog.

## Understanding Spaces

This blog is organized into multiple "spaces," each dedicated to a specific topic:

- **Machine Learning** - Deep learning, neural networks, and AI fundamentals
- **Transformers** - Attention mechanisms, LLMs, and transformer architectures
- **Web Development** - Frontend, backend, and full-stack web technologies
- **General Notes** - Miscellaneous insights and observations

## Navigation

### Homepage

Start here for an overview of all spaces and latest articles.

### Spaces

Click "Spaces" to see all available knowledge spaces.

### Individual Spaces

Each space has its own page with articles specific to that topic.

### Posts

Click any article to read the full content with formatting and code samples.

### Tags

Filter posts by topic using the tags system.

### RSS Feed

Subscribe to get the latest posts delivered to your feed reader.

## Features

✓ **Dark Mode** - Toggle the theme with the button in the navbar
✓ **Reading Time** - See estimated reading time for each post
✓ **Metadata** - Publication date and tags for every article
✓ **Breadcrumbs** - Easy navigation showing your location

## Tips

1. Use tags to discover related content
2. Check the latest articles on the homepage
3. Subscribe to RSS for updates
4. Dark mode is great for night reading

Happy exploring!
